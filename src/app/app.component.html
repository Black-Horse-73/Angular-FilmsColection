Список фильмов

Вашему вниманию предлагается веб-приложение, которое содержит список фильмов.

Каждый фильм представлен постером, названием, годом выпуска фильма в прокат.

Кроме этого, каждый фильм можно удалить нажав на кнопку с соответствующим названием

Перед списком фильмов есть форма, для добавления новых фильмов
